I learned how to utilize the Levenshtein algorithm to compare two strings and return a similarity score. I'd never used an algorithm like this before, and learning how to implement it was quite easy. Example position matching is a completely different approach to the task than the Levenshtein distance algorithm. The first difference in implementation is that the Levenshtein algorithm presents a higher degree of similarity with a lower score while the example position match algorithm presents a higher degree of similarity with a higher score.
